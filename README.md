# Live Character Counter
 # Project Description

Live Character Counter is a simple web application that allows users to type text in a textarea and view the number of characters entered in real time.
It also displays the remaining number of characters allowed and notifies the user when the maximum character limit is reached.

# Features
Real-time character counting
Displays remaining characters
Character limit set to 200
Warning message when the limit is reached
Responsive and user-friendly interface
Attractive design using HTML, CSS, and JavaScript
